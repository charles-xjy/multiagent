**8年多时间，建设范围扩至200多平方公里——**

**“一张图”看雄安新区嬗变**

![](https://mposs.bjnews.com.cn/2025/06/02/3517488378348410004_6dce72cb49c25927ea87b3af35bb5596.jpg)

《人民日报》（2025年06月01日第01版）

项目图、卫星图、规划图……探访河北雄安新区管委会，印象最深的是一张张图。

“真正把高标准的城市规划蓝图变为高质量的城市发展现实画卷。”2023年5月，习近平总书记在河北雄安新区考察时强调。

牢记习近平总书记嘱托，推动蓝图变实景，这座高水平现代化城市正在拔节生长。8年多时间，雄安新区建设范围已扩至200多平方公里。

一张图，能看出什么？

来到雄安工作4年来，雄安新区启动区管委会疏解服务处工作人员姚晨宇攒下一摞项目图——从2021版到2025版，20多张。翻开逐一浏览，像是在回放雄安城市建设纪录片：2021年，启动区建设起步不久，当时的项目图还有些空；如今工程增多、建设加快，图渐渐被填得满满当当。

发展速度，背后是改革驱动。走进正在内部装修的中国中化总部，姚晨宇拿出一张2022年新区启动区项目图，图中规划的中国中化总部大楼，如今已“长”成150米高的建筑，预计年底前具备进驻条件。按照一般流程，这个体量的工程，办齐各类施工许可需要6个月。得益于新区深化行政审批制度改革，“一枚印章管到底”，施工许可分阶段办理，项目从拿地到进场平整、土方开挖，仅用时1个月。

![](https://mposs.bjnews.com.cn/2025/06/02/3517488378348410004_f415de53a46c5b07ee4321c241e0eec5.jpg)

2024年11月4日，无人机航拍的中国星网总部办公楼。河北日报记者李畅摄

这张图，照见“快”中有“质”。2024年10月，中国星网在首批疏解央企中第一个进驻办公，并联合上下游企业发起成立中国空天信息和卫星互联网创新联盟，吸引50多家链上企业聚集，“聚”出新区第一条产业链，“链”出高质量发展新图景。

这张图，照见“质”中出“新”。项目图中，高新技术企业不断增多，梅卡曼德是其中之一。2024年11月，梅卡曼德（北京）机器人科技有限公司在雄安新区中关村科技园揭牌。随后，梅卡曼德与其他企业发起成立雄安新区中关村机器人产业联盟，推动智能机器人产业协同发展。

围绕新一代信息技术等领域，雄安新区谋划未来产业新赛道，加快培育新质生产力。目前，雄安新区中关村科技园引入企业130多家，其中国家高新技术企业35家。

点开雄安新区卫星图，一个个新项目掩映在绿色之中。300米进公园、1公里进林带、3公里进森林，新区设立以来，已累计造林48.1万亩，森林覆盖率从11%提高到如今的35%，“含绿量”逐步提高。

![](https://mposs.bjnews.com.cn/2025/06/02/3517488378348410004_0a3de32d1499b0f5fbff845e61258f98.jpg)

无人机拍摄的雄安新区白洋淀。（河北日报资料片）河北日报记者田明摄

一张蓝图干到底，规划图正在变成现实画卷。

截至目前，新区已编制《河北雄安新区规划纲要》等规划体系，从12平方公里的容东片区，到38平方公里的启动区，再到目前建设范围扩大到200多平方公里，一年接着一年干，新区规划图建设“进度条”不断更新，已新建道路800多公里，4900多栋楼宇拔地而起。

眼下，首批疏解央企中国中化、中国华能即将完成装修，第二批4家疏解央企已经选址。雄安新区启动区管委会相关负责人介绍，启动区重点项目今年已增至142个。

“今年，重点服务中国中化等疏解央企进驻办公。”登上中国中化总部21楼，姚晨宇打开2025年启动区项目图，眺望四周，附近的中国矿产总部即将封顶，一批在建项目尽收眼底……
